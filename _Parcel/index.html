Install Parcel
В новой версии 1.12.5 всё работает нормально

- поставить новую npm install -g parcel-bundler

- или нужную версию npm install -g parcel-bundler@1.12.5

- или локально npm install parcel-bundler --save-dev

Флаг --save-dev указывается, что информацию о пакете следует сохранить именно в секции devDependencies файла package.json:

- удалить старую npm uninstall -g parcel-bundler

Create a package.json

- поставить новую npm init -y

Create an index.html and index.js

Run/Build

- запуск parcel index.html

- сборка parcel build index.html

- или npm run имя_скрипта

Preferences

В Parcel по умолчанию не работает async/await. Лечится добавлением в package.json:

"browserslist": ["last 1 Chrome version"]

или каким-либо из этих способов: https://flaviocopes.com/parcel-regeneratorruntime-not-defined/
